COBA_Résultats
Présentation Historique ↓   Installation ↓
COBA_Résultats est un programme qui facilite le transfert de données entre des chiffriers (Excel, Sheets, etc.) et le portail pédagogique Coba. De plus, il est compatible avec certaines plateformes numériques telles que Classroom, Moodle et Quizizz.

La plus récente version du programme peut effectuer les transferts suivants :
  • Listes de classe de COBA  →  chiffrier  
  • Notes/Commentaires/Compétences d’un chiffrier  →  COBA  
  • Résultats de Classroom/Moodle/Quizizz  →  COBA ou chiffrier  
  • Résultats de COBA  →  chiffrier ou COBA  
Pour lancer le programme, utilisez le raccourci clavier Ctrl+Maj+Alt+V, ou double-cliquez sur l'icône située sur le Bureau. Il est également recommandé d'épingler ce raccourci à la barre des tâches de Windows pour un accès rapide.

Installation Présentation ↑   Historique ↑
Voici la procédure à suivre pour installer COBA_Résultats sur un ordinateur PC/Windows.
  1. Faire un clic-droit sur ce lien
  2. Choisir « Enregistrer le lien sous... »
  3. Choisir le Bureau comme dossier de destination
  4. Cliquer sur « Enregistrer »
  5. Selon le navigateur web utilisé, il se peut qu’un avertissement de téléchargement suspect (ou dangereux) soit affiché. Pas de panique ! Si c’est le cas :
    1. Cliquer sur l’avertissement
    1. Cliquer sur « Télécharger le fichier... »
  6. Aller sur le Bureau
  7. Repérer le dossier compressé « COBA_R_install » et faire un clic-droit dessus
  8. Choisir « Ouvrir avec » puis « Explorateur Windows »
  9. Dans la nouvelle fenêtre ouverte, double-cliquer sur « COBA_Résultats (installation) »
  10. Cliquer ensuite sur « Extraire tout »
  11. Dans la fenêtre suivante :
    1. Ne pas modifier le dossier déjà inscrit
    2. S’assurer que la case « Afficher les dossiers... » est cochée
    3. Cliquer sur « Extraire »
  12. Dans la nouvelle fenêtre ouverte, double-cliquer sur « COBA_Résultats (installation) »
  13. Si la fenêtre « Windows a protégé votre ordinateur » apparaît :
    1. Cliquer sur « Informations complémentaires »
    2. Cliquer sur « Exécuter quand même »
  14. Suivre attentivement les instructions de l’installateur qui s’exécute
  15. Si l’installation est réussie, fermer les deux fenêtres « COBA_R_install » et supprimer les deux dossiers « COBA_R_install » de votre Bureau
