Tutoriel #1
« Comment transférer une liste de classe de COBA à un chiffrier »

1)
Dans le menu gauche du portail COBA, cliquez sur Listes de classe.
2)
Dans le menu déroulant en haut de la page, sélectionnez une des Classes.
3)
Sur le clavier, appuyez sur Ctrl+A pour tout sélectionner.
4)
Ensuite, appuyez sur Ctrl+C pour copier les informations.
5)
Lancez le programme COBA_Résultats en utilisant le raccourci habituel.
6)
Suivez les instructions à l’écran. Vous pouvez choisir les informations à transférer et modifier la façon dont elles seront présentées dans votre chiffrier. Vos préférences sont enregistrées afin d'optimiser la procédure pour les groupes suivants.
7)
Lorsque le programme indique que les informations ont été traitées, ouvrez l’application de votre choix (Excel, Sheets, etc.), placez le curseur à l’endroit désiré, puis appuyez sur Ctrl+V pour coller les informations sélectionnées.

N.B. COBA_Résultats permet toutes sortes de transferts.
