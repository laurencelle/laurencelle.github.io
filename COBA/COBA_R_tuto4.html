Tutoriel #4
« Comment transférer un rapport exporté à COBA... »

« ... ou transférer un rapport exporté à un chiffrier »

1)
 Dans votre navigateur, ouvrez la page qui contient les résultats d'un ou plusieurs quiz dans un format téléchargeable. Voici les chemins à suivre selon les plateformes :
  • CommonLit → My Classes → Overall Performance → [choisir un groupe]
         → Performance by Assignment → Download to CSV
  • Formative → Responses → More options (⋮) → Export → Data : Raw scores
         → Format : Google Sheets → Simplified Export → Export
  • iDoceo → Exporter → XLS → Onglet courant [ou autre...] → Exporter
         → [déplacer le fichier dans un dossier accessible sous Windows]
  • NoRedInk → Student Data → [choisir un groupe] → Download Data → Excel
2)
 Une fois le fichier téléchargé, ouvrez-le dans Excel ou dans Sheets (selon le format).
3)Sélectionnez la plage de données qui correspond aux résultats d'un groupe complet. ATTENTION : votre sélection doit inclure les noms d'élèves et les résultats. Cela permet au programme de vérifier que l'ordre alphabétique est respecté. Si plusieurs colonnes de résultats sont sélectionnées, c'est la plus à droite qui sera traitée.
4)
 Ensuite, appuyez sur Ctrl+C pour copier les données.
5)
 Lancez le programme COBA_Résultats via le raccourci habituel.
6)
 Une fenêtre affichera la liste des élèves ainsi que leurs résultats. Profitez-en pour vérifier le nombre d'élèves, incluant les absents.
7)
 Pour transférer les résultats vers COBA, cliquez sur « COBA ».Pour transférer les résultats vers un chiffrier, cliquez sur « Ailleurs ».
8)
 Dans votre navigateur, accédez à la page souhaitée du portail COBA avant de cliquer sur OK. Ne touchez à rien pendant que les résultats sont transférés. Accédez à votre chiffrier et placez le curseur dans la bonne colonne, vis-à-vis l'élève #1, puis appuyez sur Ctrl+V pour coller les résultats.

COBA_Résultats permet toutes sortes de transferts.
