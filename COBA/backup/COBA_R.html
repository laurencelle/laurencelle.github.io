COBA_Résultats
Présentation Historique ↓   Installation ↓
COBA_Résultats est un programme qui facilite le transfert de données entre des chiffriers (Excel, Sheets, etc.) et le portail pédagogique Coba. De plus, il est compatible avec plusieurs plateformes numériques telles que Classroom, Formative, Moodle et Quizizz.

La plus récente version du programme peut effectuer les transferts suivants :
  • Listes de classe de COBA  →  chiffrier  
  • Notes/Commentaires/Compétences d’un chiffrier  →  COBA  
  • Résultats de plateformes numériques  →  COBA ou chiffrier  
  • Résultats de COBA  →  chiffrier ou COBA  
... ainsi que la suppression des données du portail COBA  

Pour lancer le programme, utilisez le raccourci clavier Ctrl+Maj+Alt+V, ou double-cliquez sur l’icône située sur le Bureau. Il est également recommandé d’épingler ce raccourci à la barre des tâches de Windows pour un accès rapide.

Pour installer COBA_Résultats sur un portable du collège ou un ordinateur de classe :
  1. Dans une session Windows, ouvrez le menu Démarrer.
  2. Tapez directement \\pdq-deploy au clavier, puis appuyez sur « Entrée ».
  3. Dans la fenêtre qui s’ouvre, double-cliquez sur le dossier COBA_Resultats.
  4. Lancez l’installateur en double-cliquant sur COBA_Résultats (installation).
  5. Suivez attentivement les instructions affichées pour finaliser l’installation.
  6. Une fois l’installation réussie, vous pouvez lancer le programme (voir Présentation).
Pour installer COBA_Résultats sur un ordinateur personnel, vous pouvez emprunter une clé USB d’installation auprès de Julien Forest-Pesant (SEC. 1), Marjolaine Balthazar (SEC. 2), Sylvie Robert (SEC. 3), Olivier Arsenault (SEC. 4), Suzie Mathieu (SEC. 5) ou Julien Laurencelle (DÉVELOPPEUR).
