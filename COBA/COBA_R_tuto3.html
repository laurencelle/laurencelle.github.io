« Comment transférer les résultats d'un quiz à COBA... »

« ... ou transférer les résultats d'un quiz à un chiffrier »

1)
Dans votre navigateur, ouvrez la page qui contient les résultats du quiz* pour un seul groupe d'élèves. Dans Formative, « développez » un seul groupe, puis réduisez le zoom de manière à voir tous les résultats du groupe. Dans Moodle, assurez-vous d'inclure dans le rapport les tentatives des utilisateurs inscrits (ayant ou non effectué le test). Dans Quizizz, faites défiler la page jusqu'à la fin avant l'étape suivante.
2)
Sur le clavier, appuyez sur Ctrl+A pour tout sélectionner.
3)
Ensuite, appuyez sur Ctrl+C pour copier les données.
4)
Lancez le programme COBA_Résultats via le raccourci habituel.
5)
Une fenêtre affichera la liste des élèves ainsi que leurs résultats. Profitez-en pour vérifier le nombre d'élèves, incluant les absents. Vous pouvez aussi modifier le format des notes en cliquant sur « Format... ».
6)
Pour transférer les résultats vers COBA, cliquez sur « COBA ».Pour transférer les résultats vers un chiffrier, cliquez sur « Ailleurs ».
7)
Dans votre navigateur, accédez à la page souhaitée du portail COBA avant de cliquer sur OK. Ne touchez à rien pendant que les résultats sont transférés. Accédez à votre chiffrier et placez le curseur dans la bonne colonne, vis-à-vis l'élève #1, puis appuyez sur Ctrl+V pour coller les résultats.

*Les plateformes suivantes sont compatibles :
 •  Classroom
 •  Formative
 •  Moodle
 •  Quizizz
Si vous utilisez d'autres plateformes (comme iDoceo), vous pouvez exporter les résultats vers Classroom afin d'utiliser le programme.

COBA_Résultats permet toutes sortes de transferts.
